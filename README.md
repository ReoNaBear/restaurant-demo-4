# Restaurant-List-demo-4-
這是AC作業中的餐廳清單 (加上了登入功能的版本)

功能: 

1.使用者可以在首頁觀看到所有餐廳 

2.使用者能夠修改餐廳資訊

3.使用者可以新增一家餐廳

4.使用者可以刪除一筆餐廳資料

5.使用者能利用查詢功能搜索到符合的餐廳
